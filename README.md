# GrowingIO Design Charts

一套基于 [G2](https://g2.antv.vision/zh) 的 Chart 组件库

![G2](https://g2.antv.vision/zh)
![License](https://img.shields.io/github/license/growingio/gio-design)

## Getting Started

安装依赖：

```
yarn install
```

启动预览网站：

```
yarn storybook
```
