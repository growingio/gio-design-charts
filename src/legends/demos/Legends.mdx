# Legend 图例

## 定义

图例解释了图表区域中包含的所有视觉元素的含义。Legend 的排放需要能紧凑且充分的利用空间。

## Legend 图例样式

- 图例与图表中对齐，且二者保持 4px 的间距。
- 每个图例之间间隔为 20px。当图例宽度超过显示区域是。其余的通过“其余 X 项”表示，Hover 通过 tooltip 显示其余 X 项的图例。距离 4px
- 线图可能需要支持虚线，所以线图和其他图的图例样式不同。
