## 主题设置

```typescript
import { DesignContext } from '@gio-design/utils';

<DesignContext.Provider value={{ theme: customTheme }}>
  <Column {...args} />
</DesignContext.Provider>;
```

## 设置主题的参数

设置 Chart 的主题，详细的方案请参考：[https://sourcegraph.com/github.com/antvis/G2/-/blob/src/theme/util/create-by-style-sheet.ts](https://sourcegraph.com/github.com/antvis/G2/-/blob/src/theme/util/create-by-style-sheet.ts)

```
const customTheme = {
  // set default colors, althought we have set default color for each legend.
  // but it's still necessary to add default colors for theme
  components: {
    axis: {
      common: {
        label: {
          offset: 16,
          style: {
            fill: '#e8ac24',
          },
        },
        line: {
          style: {
            lineWidth: 1,
            stroke: '#e8ac24',
          },
        },
      },
    },
  },
  labels: {
    style: {
      fill: '#e8ac24',
    },
  },
};




```
