# Line Chart 折线图

## 定义

Line Chart 折线图。

- 最适用：折线图用于显示数据在一个连续的时间间隔或者时间跨度上的变化，它的特点是反映事物随时间或有序类别而变化的趋势。
- 折线图时间上是由一个个的点组成，为了更好的识别趋势而将一个个的点连接成线条。点点相连的线条永远为直线，使用曲线会使趋势的变化失去真实性和精确度。

## 基础样式

- 线条宽度为 2px
- 横纵坐标的显示参考【坐标轴规范】。注：纵坐标的展示方法取决于所展示数据中的最大值/最高点。
  （可配置在图表中最高点/最低点是否标出）
- hover 时，用【信息卡片】承载具体的数值信息，并通常带有辅助线。

## 多维度拆解样式

- 通常颜色的分配根据图表中所显示的维度的数据量，从高到低排序。遵循【颜色序列】规范。
- 层级上，列表前方的项比列表后方的项层级更高。线条重叠的情况下，层级高的线条会盖住层级低的线条。
- hover 时，信息卡片承载具体数值信息。最大高度为 360px，多出则需要支持滚动。信息卡片的对齐方式依照【维度多选器】中的列表。信息卡片中的排序方式也遵循【颜色序列】。
